### 🚀 About Me

Fullstack developer based in Kuala Lumpur, currently at **Studio20**, building web and mobile apps with agentic engineering, not vibe coding.

Most of my recent work sits where product meets infrastructure: shipping Next.js and NestJS apps, wiring async job queues and AI enrichment pipelines, and owning the path from `git push` to a live container behind Nginx. I also build developer tooling, including a CLI that scaffolds production-ready Bun monorepos and a deploy tool that makes shipping the 101st project as easy as the 1st.
